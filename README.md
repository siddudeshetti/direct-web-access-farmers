# Direct Web Market Access for Farmers

A full-stack web application that connects farmers directly with buyers.

## Features

- Farmer and Buyer Registration/Login
- JWT Authentication
- Farmers can add and manage products
- Buyers can browse products
- Shopping Cart
- Place Orders
- View Order History
- Farmer can view customer orders

## Technologies

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs

## Run Locally

Install dependencies:

    npm install

Create `.env`:

    MONGO_URI=your_mongodb_connection_string
    JWT_SECRET=your_secret_key
    PORT=5000

Start the server:

    npm start

Open:

    http://localhost:5000
